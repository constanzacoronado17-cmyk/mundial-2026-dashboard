# ⚡ INICIO RÁPIDO - Dashboard a Vercel

## 3 Pasos Simples (5 minutos)

### 1️⃣ CREAR REPO EN GITHUB
```
URL: https://github.com/new
Nombre: mundial-2026-dashboard
Público: ✓ SI
→ Crear
```

### 2️⃣ SUBIR TUS ARCHIVOS
```bash
# En tu computadora, en la carpeta del dashboard:
git init
git add .
git commit -m "Inicial"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/mundial-2026-dashboard.git
git push -u origin main
```

### 3️⃣ DESPLEGAR EN VERCEL
```
URL: https://vercel.com/new
Buscar: tu repo en GitHub
Nombre: mundial-2026-dashboard
→ Deploy
```
