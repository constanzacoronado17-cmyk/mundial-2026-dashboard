# 📊 FIFA World Cup 2026 - Strategic Analysis Dashboard

Dashboard interactivo de análisis comparativo de estrategias digitales **Nike vs Adidas** para el Mundial 2026.

## 🎯 Contenido

- **Estrategia por Campaña**: Enfoques y canales clave de cada marca
- **Objetivos de Cada Campaña**: Metas específicas de Nike y Adidas
- **Tabla Comparativa de KPIs**: Métricas detalladas con gráficos interactivos
- **Análisis Detallado e Insights**: Hallazgos clave sobre rendimiento
- **Propuestas de Mejoras Estratégicas**: Recomendaciones accionables

## ✨ Características

- ✅ Diseño responsive (funciona en móvil, tablet y desktop)
- ✅ Gráficos interactivos con Chart.js
- ✅ Animaciones suaves y profesionales
- ✅ Navegación smooth scroll
- ✅ Paleta de colores moderna (Rojo/Azul/Verde Lima)
- ✅ Totalmente accesible desde cualquier dispositivo

1. **Fork este repositorio** en GitHub

3. Conecta tu cuenta de GitHub
4. Selecciona este repositorio
5. Click en "Deploy" ✅
